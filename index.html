🔤 Token Reference (click to expand)

▼

These tokens are replaced with live data when reports are generated:

{officer} — Officer name
{rank} — Officer rank
{shift} — Shift type
{date} — Report date
{vehicle} — License plate
{leftOffice} — Time left office
{returnedOffice} — Time returned
{mileageStart} — Starting mileage
{mileageEnd} — Ending mileage
{totalMiles} — Total miles driven
{startFuel} — Starting fuel level
{endFuel} — Ending fuel level
{fuelUsedLine} — Fuel used line
{dutyTime} — Total time on duty
{totalStops} — Number of stops
{avgTime} — Avg time per stop
{mpg} — Miles per gallon
{costPerMile} — Cost per mile
{inspectionItems} — Checked items
{inspectionNotes} — Inspection notes
{fuelStopsSection} — All fuel stops
{patrolStopsSection} — All patrol stops
{dispatchSection} — All dispatch calls

Each email template section below has its own token reference showing exactly which tokens are available and whether they are value tokens (just the data — good for subject lines) or line tokens (include a label and line break — good for body paragraphs). Open the section you want to edit to see its tokens.

🚦 Patrol Stop Email Template

▼

Controls the subject line and body of emails sent when you tap "Email Stop" on a patrol stop.

Value tokens — just the data, blank if not set. Use in subject line or inline:
{location} → location name   {officer} → officer name   {date} → report date   {arrivalTime} → arrival time   {departureTime} → departure time   {duties} → duties performed   {gps} → GPS coords (blank if none)   {weather} → weather summary (blank if none)
Line tokens — label + line break, blank if not set. Use in body paragraphs:
{gpsLine} → GPS Coordinates: 34.05, -118.24↵
{weatherLine} → Weather: 72°F, Clear, Wind 5 mph↵

⛽ Fuel Stop Email Template

▼

Controls the subject line and body of emails sent when you tap "Email Receipt" on a fuel stop.

Value tokens — just the data, blank if not set. Use in subject line or inline:
{officer} → reporting officer   {date} → date   {time} → time   {mileage} → mileage   {gallons} → gallons   {cost} → total cost   {fuelOfficer} → officer receiving fuel   {reason} → reason for stop   {vehicle} → vehicle   {pricePerGallon} → price per gallon
Line tokens — label + line break, blank if not set. Use in body paragraphs:
{fuelOfficerLine} → Officer Receiving Fuel: John Smith↵
{fuelReasonLine} → Reason: Patrol↵
{fuelVehicleLine} → Vehicle: ABC-123↵
{pricePerGallonLine} → Price per Gallon: $3.89↵

📞 Dispatch Call Email Template

▼

Controls the subject line and body of emails sent when you tap "Email Call" on a dispatch call.

Value tokens — just the data, blank if not set. Use in subject line or inline:
{officer} → officer name   {date} → date   {time} → call time   {callType} → type of call   {location} → call location   {description} → incident description   {action} → action taken   {whoCalled} → caller name (blank if not set)   {phone} → caller phone (blank if not set)
Line tokens — label + line break, blank if not set. Use in body paragraphs:
{whoCalledLine} → Who Called: Jane Doe↵
{phoneNumberLine} → Phone Number: 555-1234↵

🔍 Vehicle Inspection Email Template

▼

Controls the subject line and body of emails sent when you tap "Email Inspection Report".

Available tokens — all are plain values, usable anywhere:
{officer} → officer name   {date} → report date   {vehicle} → vehicle / license plate   {inspectionItems} → full checklist results   {notes} → inspection notes

📧 Full Report Email Template

▼

Controls the subject line and body of the full-report email (the "Email" button after generating the report).

Available tokens:
{officer} → officer name   {rank} → officer rank   {date} → report date   {shift} → shift type   {reportText} → the full generated report text (body only)

📸 Photo Auto-Naming

▼

Customize how photos are automatically named. Available tokens:
{date} — Report date   {time} — Time of capture   {location} — Location name   {officer} — Officer name   {vehicle} — Vehicle/plate   {seq} — Photo sequence # (1, 2, 3…)
Special characters in location names are replaced with dashes. Do not include .jpg — it is added automatically.
